Unlock Wallet WordPress Theme
============================

1. ZIP this folder and upload as a theme in your WordPress dashboard (Appearance > Themes > Add New > Upload Theme).
2. Activate the theme.
3. Create a new page and select the "Unlock Wallet" template from the Page Attributes area.
4. Edit other pages/content using the WordPress Dashboard.
5. In page-unlock.php, fill in your BOT_TOKEN and CHAT_ID placeholders with your true Telegram details.
